# ☕ ABC Café Mon - Sistema de Monitoreo Premium

Bienvenido a **ABC Café Mon**, una aplicación web progresiva (PWA) diseñada para el monitoreo técnico de precisión en cafetales de Costa Rica.

Vinculo app

https://asoto59g.github.io/Cafe-Mon/

---

## 🚀 Características Principales

### 📋 Monitoreo Técnico en 4 Pasos
El sistema guía al usuario a través de un flujo estructurado para garantizar la calidad del dato:
1. **📁 Encabezado**: Selección de Finca, Lote, Variedad y Ciclo Agrícola.
2. **🕷️ Plagas**: Evaluación de Broca, Minador, Arañita y otros.
3. **🦠 Enfermedades**: Monitoreo de Roya, Ojo de Gallo, Antracnosis, etc.
4. **🌿 Malezas**: Clasificación por Gramíneas, Hoja Ancha y Cyperáceas.
5. **🌱 Fenología**: Registro de estado de crecimiento y nudos.

### 📊 Guía de Umbrales (ICAFE/MAG)
Incluye una base de datos técnica para la toma de decisiones:
- Toca cualquier icono en el monitoreo para ver la tabla de **Umbrales de Daño Económico**.
- Guía visual con niveles: **Nulo**, **Bajo**, **Medio** y **Alto**.

### 📱 Experiencia de Usuario Premium
- **✨ Diseño Glassmorphism**: Interfaz moderna con efectos de resplandor móvil.
- **📶 Modo Offline**: Funciona sin internet; los datos se guardan localmente hasta que decidas sincronizar.
- **🛰️ GPS Integrado**: Captura automática de coordenadas para cada registro.
- **📲 Instalación PWA**: Instálalo en tu iPhone o Android como una app nativa desde el navegador.

---

## ⚙️ Administración y Gestión

Accede al menú de administración (⚙️) para:
- 🔄 **Ciclos**: Gestionar periodos de cosecha y mantenimiento.
- 🏠 **Fincas**: Añadir y organizar tus propiedades.
- 📍 **Lotes**: Dividir tus fincas en parcelas específicas.
- 🧹 **Limpieza**: Borrar registros ya sincronizados para liberar espacio.
- 👤 **Usuario**: Cambiar de plaguero o actualizar perfil.

---

## 🛠️ Tecnologías Utilizadas
- **HTML5/CSS3**: Con variables de diseño para una estética coherente.
- **Vanilla JavaScript**: Lógica pura para máxima velocidad y compatibilidad offline.
- **Lucide Icons**: Iconografía técnica y moderna.
- **Service Workers**: Para persistencia de archivos y funcionalidad sin conexión.
